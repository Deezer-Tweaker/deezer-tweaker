## Features

* Plugins
* Themes
* An in-app marketplace to add/remove plugins as desired
* Custom CSS
* Enable DevTools

## Installing

Go to the [latest release](https://github.com/Deezer-Tweaker/deezer-tweaker/releases/latest). Download and run the `.exe` file.
